AseaFaceTracker

Installation
	Put the repostory in your addons folder under openFrameworks so it looks like this: /path_to_of/addons/ofxAseaRobot/. The simpleAseaExample shows how to interface the robot.

Requirements
	External addons:
		ofxMSAInteractiveObject
		ofxMSAInterpolator
		ofxMSACore
		ofxCv
		ofxFaceTracker
		ofxAseaRobot
	Core addons:
		ofx3DModelLoader
		ofxXmlSettings
